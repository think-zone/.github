# Think-Zone 🧠

**Local-first AI infrastructure for creators, builders, and indie makers.**

We build open-source tools that let you run powerful AI workflows entirely on your own machine — no cloud lock-in, no subscriptions, no surveillance.

---

## What We're Building

| Project | Status | Description |
|---------|--------|-------------|
| [creator-os](https://github.com/think-zone/creator-os) | 🔨 Active | ThinkPad automation stack — local AI with WSL2, Ollama, Docker, and WebEngine builder |
| [cognitive-mcp](https://github.com/think-zone/cognitive-mcp) | 🚀 v0.1.0 | MCP server framework — persistent agent memory and cognitive tools for LLM workflows |

---

## Our Philosophy

> **Own your stack. Own your data. Own your AI.**

Most AI tools rent you access to someone else's compute. We think that's backwards for builders. Think-Zone is built on the belief that the best AI infrastructure is the kind you run yourself — fast, private, and fully under your control.

---

## Tech Stack

- **Runtime:** WSL2 + Ubuntu, Docker, Ollama
- **Languages:** TypeScript, Python, Bash
- **AI Layer:** Local LLMs (Llama, Mistral, Qwen), Claude via MCP
- **Automation:** n8n, GitHub Actions, custom WebEngine workflows
- **Protocol:** Model Context Protocol (MCP) for agent tooling
